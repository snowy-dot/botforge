❓ Frequently Asked Questions

🤔 My bot won't come online, what do I do?

Try these fixes in order:

  1. Check your token (no extra spaces!)
  2. Enable "Message Content Intent" in Discord
  3. Make sure code uses process.env.DISCORD_TOKEN
  4. Check logs for error messages
  5. Verify bot is invited to a server
